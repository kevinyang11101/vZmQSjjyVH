# 宠物食品安全性研究管理系统 java971

## 项目概述

宠物食品安全性研究管理系统是基于大数据的Web系统，采用前后端分离的模式，致力于提高宠物食品的安全性和研究效率。

## 技术栈

### 开发语言
- Python

### 框架
- 后端：Django
- 前端：Vue.js（2.X）、CSS3、HTML5、JavaScript

### 开发工具
- PyCharm
- Visual Studio Code
- HbuildX

### 数据库
- MySQL 5.7.26
- 数据库管理工具：phpstudy/Navicat、phpstudy/sqlyog

## 功能模块

系统主要包含以下功能模块：
1. 宠物食品信息管理
2. 安全性数据采集与分析
3. 系统用户权限管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python版本：3.0及以上
- MySQL版本：5.7.26

## 部署步骤

1. 环境准备：安装Python、MySQL等依赖环境。
2. 数据库配置：使用提供的SQL文件初始化数据库。
3. 系统部署：配置系统参数，运行系统。

## 目录结构

```
项目根目录
│
├── backend                # 后端代码目录
├── frontend               # 前端代码目录
└── database               # 数据库相关文件目录
    ├── init.sql           # 数据库初始化SQL文件
    └── ...
```

## 核心亮点

1. 基于MVVM体系结构，前后端分离，易于维护与扩展。
3. 采用MySQL数据库，数据存储可靠。
